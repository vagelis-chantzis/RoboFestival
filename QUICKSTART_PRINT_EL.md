# Robofestival Tournament - Εκτυπώσιμο Γρήγορο Checklist (A4)

Ημερομηνία: ____________  |  Χώρος: ____________  |  Υπεύθυνος: ____________

## Προετοιμασία πριν την εκδήλωση

- [ ] Άνοιγμα του `index.html` στον browser
- [ ] Επιλογή γλώσσας (`EN / EL`)
- [ ] Επιλογή θέματος (Dark / Light)
- [ ] Επιβεβαίωση ότι ανοίγει στην καρτέλα `Participants + teams`
- [ ] Προαιρετικά: εισαγωγή προηγούμενης συνεδρίας με `Import Backup`

## Συμμετέχοντες

- [ ] Προσθήκη συμμετεχόντων (ένας-ένας ή με bulk paste)
- [ ] Έλεγχος badge πλήθους συμμετεχόντων
- [ ] Αφαίρεση λανθασμένων ονομάτων
- [ ] Επιβεβαίωση ότι όλοι οι απαιτούμενοι συμμετέχοντες υπάρχουν

## Ομάδες

- [ ] Ορισμός `Number of teams`: ______
- [ ] Πάτημα `Create randomly`
- [ ] Προσαρμογή αναθέσεων αν χρειάζεται (drag/drop ή move controls)
- [ ] Έλεγχος ότι ο πάγκος (unassigned) είναι σωστός
- [ ] Προαιρετικά: `Lock participants`
- [ ] Προαιρετικά: `Lock teams`

## Εκτέλεση διαγωνισμού (ανά επίπεδο)

Επίπεδο #: ______

- [ ] Πάτημα `Create next level`
- [ ] Ανακοίνωση ζευγαριών
- [ ] Συμπλήρωση όλων των σκορ για το τρέχον επίπεδο
- [ ] Επιβεβαίωση ότι η κατάσταση επιπέδου είναι complete
- [ ] Επανάληψη για επόμενο επίπεδο

Σημειώσεις:

- Με την έναρξη του πρώτου επιπέδου, συμμετέχοντες/ομάδες κλειδώνουν αυτόματα.
- Με μονό αριθμό ομάδων, μία ομάδα παίρνει `BYE`.

## Ολοκλήρωση τουρνουά

- [ ] Πάτημα `End tournament`
- [ ] Άνοιγμα `Leaderboard` και έλεγχος κατάταξης
- [ ] Πάτημα `Export Excel`
- [ ] Πάτημα `Export Backup` (αποθήκευση JSON)
- [ ] Επιβεβαίωση ότι αποθηκεύτηκαν και τα 2 αρχεία

## Αρχειοθέτηση μετά την εκδήλωση

- [ ] Αποθήκευση αρχείου αποτελεσμάτων Excel
- [ ] Αποθήκευση αρχείου backup JSON
- [ ] Προαιρετικά: screenshot της τελικής κατάταξης
- [ ] Reset tournament μόνο αφού επιβεβαιωθεί ότι τα αρχεία έχουν σωθεί

## Γρήγορη αντιμετώπιση προβλημάτων

- Το Excel είναι ανενεργό -> κάντε πρώτα end tournament.
- Δεν επιτρέπεται επεξεργασία συμμετεχόντων/ομάδων -> ενεργό lock ή auto-lock μετά το πρώτο επίπεδο.
- Χάθηκαν δεδομένα συνεδρίας -> χρήση `Import Backup`.
- Έγινε λάθος import -> reset και εισαγωγή του σωστού JSON.
