# Robofestival Tournament - Quick Start

Use this page during the event for fast operation.

## 1) Start the app

1. Open `index.html` in your browser.
2. Check top bar:
   - Language: `EN / EL`
   - Theme: Dark/Light
3. Confirm you are in `Participants + teams` tab.

## 2) Add participants

- Add one by one in `Participant name` + `Add`, or
- Paste many names in `Bulk paste` (one per line) + `Add lines`.
- Verify total from the badge next to `Participants`.

## 3) Create teams

1. Set `Number of teams`.
2. Click `Create randomly`.
3. Optional adjustments:
   - Drag players between teams
   - Drag to `Unassigned (bench)` if needed
4. (Optional) Click `Lock teams` and `Lock participants` before starting.

## 4) Run levels

1. Go to `Levels + scoring`.
2. Click `Create next level`.
3. Enter scores for all matches.
4. Repeat `Create next level` for each new level.

Important:

- After the first level, team roster moves are locked by default, but you can unlock participants anytime to edit the roster again (confirm when prompted).
- If odd number of teams, one team gets `BYE`.

## 5) Finish tournament

1. Click `End tournament`.
2. Go to `Leaderboard` to show final ranking.
3. Click `Export Excel` to download results.

## 6) Save and safety

- Click `Export Backup` to download full session JSON.
- Use `Import Backup` to restore on this or another device.
- Use `Reset tournament` only when you want a fresh start.

## Common fixes

- **Excel export disabled:** end tournament first.
- **Cannot edit teams/participants:** check `Lock participants` / `Lock teams` and hints; unlock participants if you intentionally need roster changes during levels.
- **Data missing after browser issue:** use `Import Backup`.
