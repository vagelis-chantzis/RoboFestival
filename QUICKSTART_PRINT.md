# Robofestival Tournament - Print Quick Checklist (A4)

Date: ____________  |  Venue: ____________  |  Operator: ____________

## Pre-Event Setup

- [ ] Open `index.html` in browser
- [ ] Select language (`EN / EL`)
- [ ] Select theme (Dark / Light)
- [ ] Confirm app opens in `Participants + teams`
- [ ] Optional: import previous session with `Import Backup`

## Participants

- [ ] Add participants (single or bulk paste)
- [ ] Verify participant count badge
- [ ] Remove any incorrect names
- [ ] Confirm all required participants are listed

## Teams

- [ ] Set `Number of teams`: ______
- [ ] Click `Create randomly`
- [ ] Adjust assignments if needed (drag/drop or move controls)
- [ ] Check unassigned bench is correct
- [ ] Optional: `Lock participants`
- [ ] Optional: `Lock teams`

## Run Competition (per level)

Level #: ______

- [ ] Click `Create next level`
- [ ] Announce pairings
- [ ] Enter all scores for current level
- [ ] Confirm level status shows complete
- [ ] Repeat for next level as needed

Notes:

- After levels start, use `Unlock participants` if you need roster edits mid-event (confirm when prompted).
- Odd number of teams -> one `BYE`.

## Finish Tournament

- [ ] Click `End tournament`
- [ ] Open `Leaderboard` and verify rankings
- [ ] Click `Export Excel`
- [ ] Click `Export Backup` (save JSON)
- [ ] Confirm both files saved

## Post-Event Archive

- [ ] Store Excel results file
- [ ] Store JSON backup file
- [ ] Optional: screenshot final leaderboard
- [ ] Reset tournament only when all files are confirmed saved

## Troubleshooting (Quick)

- Excel button disabled -> end tournament first.
- Cannot edit participants/teams -> check lock buttons; unlock participants if allowed for your event rules.
- Session lost -> use `Import Backup`.
- Imported wrong file -> reset and re-import correct JSON.
